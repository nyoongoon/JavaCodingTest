## 재귀 함수 
### if 문을 사용하여 **종료조건**을 설정!

``` java
  if(종료조건){
	    return; //함수 종료의 의미
	}else{ //종료조건이 아닐때
	    실행문
	}
```  

### 재귀함수는 스택프레임을 사용한다.

              |	             |
              | (<메소드>	   )|	
              | (매개변수 정보  )|
              | (지역변수 정보  )|
              | (복귀 주소	   )|-------- 
              |( ㄴ==호출 라인  )|       |
              |(...	     )|       |
              |---------------|       |
              | (<메소드>	    )|       |	
              | (매개변수 정보   )|       |
              | (지역변수 정보   )|       |
              | (복귀 주소	    )| <—---- 아래 스택(먼저 호출된)의 메소드로 복귀 (백트래킹)
              | ( ㄴ==호출 라인 )|
              | (  …	     )|
              |---------------|
                  <스택 프레임>

- 1. 무방향 그래프
 
                                      graph[a][b]   
              ① ㅡ ② ㅡ ⑤      1, 2                 ①  ②  ③  ④  ⑤

- 2. 방향 그래프
 
                                      graph[a][b]  
              ① -> ② -> ⑤    1, 2                 ①  ②  ③  ④  ⑤

              ③ -> ④          3, 4               ② 0  0   0  0  1
